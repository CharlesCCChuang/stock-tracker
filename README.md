# 美股回檔追蹤器

依類股分組追蹤 76 檔美股的「連續下跌」最大跌幅與目前回檔，每個交易日盤後自動更新。

## 連續下跌定義

- 從段內高點開始下跌，只要沒出現「反彈」就屬同一段連續下跌。
- **反彈** = 下跌途中上漲超過（段內高點＋段內低點）÷ 2 的中點，盤中碰到即成立。
- 反彈成立後該段結束歸零，之後以反彈後的新高點重新起算下一段。
- 高點、低點皆採影線（盤中極值）；「目前回檔」以最新收盤價計算。

## 架構

- `tracker.py` — 從 Yahoo Finance 抓兩年日線，逐日狀態機計算，輸出 `data.json`
- `index.html` — 靜態頁面（GitHub Pages），依 10 大類股分組顯示，可切換依回檔深度排序
- `.github/workflows/update.yml` — 台北時間週二～週六早上 05:30 自動更新
